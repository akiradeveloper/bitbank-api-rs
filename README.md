# bitbank-api-rs

Rust library for [bitbank API](https://github.com/bitbankinc/bitbank-api-docs).

## Features

- Async support
- Public API
- Stream API
- (TODO) Private API

## Install

`cargo add bitbank-api`

## Disclaimer

This is not an official library from bitbank.
